```

```

1. Acesse o painel do InfinityFree
2. Clique em "MySQL Databases"
3. Clique no phpMyAdmin ao lado do seu banco de dados
4. Verifique se todas as tabelas foram criadas corretamente

## Solução de Problemas

### Erro de conexão com o banco de dados:
- Verifique se as credenciais no arquivo `backend/config.php` estão corretas
- Verifique se o banco de dados foi criado corretamente

### Erro 404 (Página não encontrada):
- Verifique se o arquivo `.htaccess` foi enviado corretamente
- Verifique se os links nas páginas estão apontando para o domínio correto

### Permissão negada para upload de arquivos:
- Verifique as permissões da pasta `uploads` (deverá ser 755 ou 777)
- Crie manualmente as subpastas necessárias (ex: `uploads/perfil`, `uploads/artigos`)

### Imagens não são exibidas:
- Verifique os caminhos das imagens nos arquivos HTML/CSS
- Certifique-se de que as imagens foram enviadas para as pastas corretas

### Erros PHP:
- Acesse o arquivo `phpinfo.php` para verificar a configuração do PHP no servidor
- Entre em contato com o suporte do InfinityFree se encontrar limitações nas configurações
